# Archived development notes

These files document earlier development sessions, one-off fix logs, and
prototype-stage design docs. They are kept for historical reference only;
the current behavior of the project is defined by the code and by the
top-level `README.md` / `CHANGELOG.md`.

Anything in here may be stale.

- `dev-scripts/` — ad-hoc Python probes used during development. Not part
  of the regular test suite (`tests/` at the repo root is).
